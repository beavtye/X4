# ChatGPT Apps Setup Checklist

A replacement ChatGPT Plus account should connect these apps:

## Gmail

- Connect the dedicated bridge mailbox
- Confirm search can find mail from `mimobot@163.com`
- Confirm the latest X4BRIDGE report is readable

## GitHub

- Install the ChatGPT connector for the GitHub account
- Authorize only the required repository when possible
- Confirm `beavtye/X4` is visible
- Confirm the repository is Private

## Linear

- Connect the `Mantxi` workspace
- Open the `X4Bot` project
- Confirm open issues and project documents are readable

## Continuity test

Use the startup prompt in `GPT_BOOTSTRAP.md`. The new account should produce a status summary without relying on old chat history.
