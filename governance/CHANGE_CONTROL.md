# Change Control

## Framework-owned changes

The following require ChatGPT review before implementation:

- New top-level directories
- New project roots
- Cross-module file moves
- Repository-wide naming changes
- Deployment and handoff structure changes
- Changes to project manifests

## Implementation changes

Mimo may implement within an approved project path when the task names the target directory and acceptance criteria.

## Evidence required

Each completion report must include:

- Files added or changed
- Tests run and results
- Runtime impact
- Known risks
- Rollback method
- Any requested structural changes

## Protected facts

- GitHub is the source of truth for code and framework documents.
- Linear is the source of truth for open work and approvals.
- Gmail is the source of truth for the latest Mimo handoff and runtime evidence.
